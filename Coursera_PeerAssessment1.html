Strategy used is to replace occurring NA with the averaged value for that interval across all dates.

This is achieved by going through all rows using a FOR loop. Then, for all rows where there is NA for steps, we replace it with the value of steps from intevalsteps data calculated before, making sure that it is for the relevant Time interval by finding a matching value of the row 1 value of interval and the interval from activity of corresponding NA.

for (i in 1:nrow(activity) )  #Loop 'i' : Go from row 1 to end of Data.frame
  {
        if( is.na(activity[i,1]) )  # Check the value in (Row 'i', Column 1) of activity data frame, TRUE ?
            {
                activity[i,1] <- intervalsteps [
                                       which(activity[i,3]==intervalsteps$interval),2
                                      ]
            # replace the NA, with the value found in 
      #intervalsteps[where row 1 value is same as interval of the activty of the correponding NA, col 2 ]
            }
    }

4.1 Are there differences in activity patterns between weekdays and weekends?

we first convert the date in activity data frame from Character to Date data type. Then add a new column called Day in data frame which is filled with the relevant day against the date using the Weekdays() function.

activity$date <- as.Date(activity$date, "%Y-%m-%d")
activity$day <- weekdays(activity$date)
